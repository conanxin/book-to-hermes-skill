# Roadmap

## Current Stable Version

**v0.1.0-local**

Supported formats:
- Markdown (.md)
- Plain text (.txt)
- Text-based PDF (.pdf)
- HTML (.html)
- DOCX (.docx)
- EPUB (.epub)

## Near Term

- [ ] Continue daily use with real documents
- [ ] Build 5–10 formal book skills in personal library
- [ ] Improve manual glossary enrichment workflow
- [ ] Use bilingual glossary workflow for paired English / Chinese texts
- [ ] Document edge cases from real-world usage

## Mid Term

- [ ] Personal Reading Memex Index — cross-skill topic navigation
- [ ] Cross-document topic index — find related concepts across skills
- [ ] People / concept / source maps — entity relationship graphs
- [ ] Writing packets from multiple skills — synthesize across sources
- [ ] Translation comparison workflow — side-by-side bilingual reading
- [ ] Skill inventory dashboard — overview of all generated skills

## Optional Future Work

- [ ] RTF support via `striprtf`
- [ ] MOBI / AZW support via Calibre `ebook-convert`
- [ ] Optional LLM enrichment pipeline (post-generation, user-approved)
- [ ] Paragraph-level chapter alignment for bilingual texts
- [ ] Public example dataset after data review

## Not Planned for Now

- OCR for scanned PDFs
- DRM EPUB handling
- Automatic online fetching
- Automatic LLM enrichment by default
- Automatic dependency installation
- Cloud sync or multi-user sharing
