## If Publishing Publicly

Please select one of the following licenses before public release:

### Recommended Options

1. **MIT License**
   - Permissive
   - Allows commercial use, modification, distribution
   - Requires attribution
   - Simple and widely understood

2. **Apache License 2.0**
   - Permissive
   - Explicit patent grant
   - Requires attribution
   - Good for tools/utilities

3. **GPL-3.0 or AGPL-3.0**
   - Copyleft
   - Requires derivative works to be open source
   - Choose if you want to ensure the tool stays open

### Not Recommended

- **BSD-2-Clause / BSD-3-Clause**: Fine but less explicit about patents
- **CC0 / Unlicense**: Not recommended for software
- **Proprietary / All Rights Reserved**: Not suitable for public GitHub repo

## How to Add a License

1. Choose a license from https://choosealicense.com/
2. Create a `LICENSE` file in the repository root
3. Copy the full license text
4. Update the copyright year and name
5. Commit the file
